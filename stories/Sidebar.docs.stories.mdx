## Key props

- `menuItems`: hierarchical navigation schema with optional badges and children.
- `collapsed` + `onToggle`: control layout width and toggle button state.
- `profileSectionProps`, `profileButtonProps`, `logoutButtonProps`: configure the footer actions.
