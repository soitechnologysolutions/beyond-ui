# DashboardLayout

`DashboardLayout` stitches together the persistent `Sidebar` and `DashboardHeader` components to provide a full-page application shell. Import the bundled CSS once in your host app to ensure all semantic tokens resolve:

<Source
  lang="tsx"
  code={`import '@beyondcorp/beyond-ui/dist/styles.css';`}
/>

Use the layout to compose analytics dashboards, admin tools, or internal platforms where a collapsible navigation rail is required.

## Key props

- `sidebarMenuItems`: hierarchical menu model rendered inside the sidebar.
- `activeSidebarItem` + `onSidebarItemClick`: control the highlighted navigation node.
- `dashboardHeaderProps`: pass search, breadcrumb, and action content to the header.
